System Architecture

Hardware

Heltec LoRa 32 V3
  • ESP32 dual-core @ 240 MHz
  • SX1262 LoRa radio chip onboard
  • 0.96" OLED display
  • Built-in LED, USB-C, onboard button
  • Battery management + solar charging input
  • ~$25/board — no soldering needed
Two boards, one protocol
  • START board: trigger (button or sensor) → sends LoRa packet
  • FINISH board: receives packet → starts timer → on trigger: calculates elapsed time, sends result via LoRa + HTTPS
